# Tracker — Protótipo funcional do aluno

Este protótipo foi feito em HTML, CSS e JavaScript puro, com foco na experiência do aluno.

## Arquivos
- `index.html`
- `styles.css`
- `script.js`

## Como usar
1. Abra o arquivo `index.html` no navegador.
2. Na tela de login, clique em **Entrar**.
3. Navegue entre:
   - Dashboard do aluno
   - Tela de iniciar tarefa
   - Tela de tarefa em andamento
   - Tela de tarefa concluída

## O que está funcional
- Navegação entre as telas
- Login simulado
- Dashboard com cards e gráfico semanal
- Abertura do detalhe da tarefa
- Início da tarefa
- Cronômetro simples
- Pausa da sessão
- Finalização da tarefa
- Relatório em alerta

## Observação
É um protótipo front-end. Ainda não possui banco de dados nem autenticação real.
